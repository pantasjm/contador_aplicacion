La solución del Retail

El Proyecto

Cuando una empresa del sector retail busca un nuevo local para su cadena, siempre evalúa el flujo de personas que pasan frente al local.

Este cálculo se realiza de manera manual y toma mucho tiempo, dado que no se cuenta con la cantidad necesaria de personal para evaluar todos los puntos en el mapa.

Nuestra propuesta

Mediante el uso de un sensor Arduino elaboramos un aplicativo que permite contar el número de personas que cruzan una ubicación específica, así como otros indicadores de interés para la toma de decisiones.

¿Qué necesitamos para su realización?

Internet or Things

Sensor infrarrojo que detecta el cruce de una persona, sensor de humedad y temperatura.

Web development

Usamos R Studio para la visualización de la información y la representamos en una página web estática con HTML, CSS y Javascript.
